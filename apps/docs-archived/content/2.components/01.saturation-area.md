---
title: Saturation Area
description: 2D saturation/lightness picker area
---

A 2D area component for selecting saturation and lightness values. Users can click, drag, or use keyboard navigation to pick colors.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `colorFormat` | `'hsl' \| 'hsv'` | `'hsl'` | The color model used for the gradient and value calculation |

## Keyboard Navigation

- **Arrow keys**: Move the thumb by 1 step
- **Shift + Arrow keys**: Move the thumb by 10 steps

## Accessibility

The component uses proper ARIA attributes:
- `role="group"` on the container
- `role="slider"` on the thumb
- `aria-valuenow`, `aria-valuemin`, `aria-valuemax` for current value
- `aria-valuetext` announces saturation and lightness percentages

---
title: Color Preview
description: Displays the current color
---

A visual preview component that displays the currently selected color. Shows a transparency checkerboard pattern when alpha is less than 1.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | - | Custom CSS class |
| `splitPreview` | `boolean` | `true` | When true, shows transparency pattern on half of the preview |

## Accessibility

The component uses proper ARIA attributes:
- `role="img"` to indicate it's a visual element
- `aria-live="polite"` for announcing color changes
- `aria-label` displays the current HSL value

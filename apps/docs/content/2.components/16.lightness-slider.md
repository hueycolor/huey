---
title: Lightness Slider
description: Lightness channel slider
---

A slider component for controlling the lightness channel. Displays a gradient from black through the current color to white.

## Value Range

- **Min**: 0 (black)
- **Max**: 100 (white)
- **Step**: 1

## Accessibility

- `aria-label` defaults to `'Lightness slider'`
- Keyboard navigation with arrow keys

title: Hex Input
description: Hex color input field
---

An input component for entering and editing hex color values. Supports keyboard navigation for incrementing/decrementing color values.

## Features

- Validates hex input on blur and enter
- Supports 6-character hex values
- Smart pair selection for keyboard adjustments

## Keyboard Navigation

- **Up/Down arrows**: Increment/decrement the selected hex pair(s) by 1
- **Shift + Up/Down**: Increment/decrement by 10
- **Enter**: Confirm the value

Select specific hex pairs (RR, GG, BB) to adjust only those values.

## Behavior

- On focus, the entire value is selected
- Invalid hex values are reverted to the previous valid value
- Updates the color context in real-time

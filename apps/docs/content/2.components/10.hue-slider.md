---
title: Hue Slider
description: Hue channel slider
---

A slider component for controlling the hue channel. Displays a rainbow gradient representing the full color spectrum from 0 to 360 degrees.

## Value Range

- **Min**: 0
- **Max**: 360
- **Step**: 1

## Accessibility

- `aria-label` defaults to `'Hue slider'`
- Keyboard navigation with arrow keys

---
title: Color Swatch
description: Color swatch element for preset colors
---

A component that displays a list of preset colors that users can select from. Useful for providing quick access to commonly used colors.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `swatch` | `string[]` | - | Array of color strings (hex, rgb, hsl, etc.) |
| `aria-label` | `string` | `'Color swatch'` | Accessible label for the swatch |

## Usage

```vue
<ColorSwatch :swatch="['#ff0000', '#00ff00', '#0000ff']" />
```

## Keyboard Navigation

- **Enter/Space**: Select the focused color

## Accessibility

The component uses proper ARIA attributes:
- `role="listbox"` on the container
- `role="option"` on each color
- `aria-selected` indicates the currently selected color

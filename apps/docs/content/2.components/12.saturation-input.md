---
title: Saturation Input
description: Saturation channel input field
---

An input component for controlling the saturation channel of the HSL color model. Accepts values from 0 to 100 percent.

## Value Range

- **Min**: 0 (grayscale)
- **Max**: 100 (fully saturated)

## Keyboard Navigation

- **Up/Down arrows**: Increment/decrement by 1
- **Shift + Up/Down**: Increment/decrement by 10
- **Enter**: Confirm the value

## Behavior

- Invalid values are reverted to the previous valid value
- Values are clamped to the 0-100 range

---
title: Hue Input
description: Hue channel input field
---

An input component for controlling the hue channel of the HSL color model. Accepts values from 0 to 360 degrees.

## Value Range

- **Min**: 0
- **Max**: 360

## Keyboard Navigation

- **Up/Down arrows**: Increment/decrement by 1
- **Shift + Up/Down**: Increment/decrement by 10
- **Enter**: Confirm the value

## Behavior

- Invalid values are reverted to the previous valid value
- Values are clamped to the 0-360 range

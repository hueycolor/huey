---
title: RGB Inputs
description: Red, green, and blue channel input fields
---

Input components for controlling individual RGB color channels. Each accepts values from 0 to 255.

## Components

- `RedInput` - Red channel
- `GreenInput` - Green channel
- `BlueInput` - Blue channel

## Value Range

- **Min**: 0
- **Max**: 255

## Keyboard Navigation

- **Up/Down arrows**: Increment/decrement by 1
- **Shift + Up/Down**: Increment/decrement by 10
- **Enter**: Confirm the value

## Behavior

- Invalid values are reverted to the previous valid value
- Values are clamped to the 0-255 range
- Automatically updates hue, saturation, and lightness values

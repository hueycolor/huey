---
title: Alpha Slider
description: Alpha channel slider
---

A slider component for controlling the alpha (opacity) channel. Displays a gradient from transparent to opaque with a checkerboard pattern to indicate transparency.

## Value Range

- **Min**: 0 (fully transparent)
- **Max**: 1 (fully opaque)
- **Step**: 0.01

## Accessibility

- `aria-label` defaults to `'Alpha slider'`
- Keyboard navigation with arrow keys

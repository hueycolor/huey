---
title: HSL Inputs
description: Hue, saturation, and lightness channel input fields
---

Input components for controlling individual HSL color channels.

## Components

| Component | Range | Description |
|-----------|-------|-------------|
| `HueInput` | 0-360 | Hue in degrees |
| `SaturationInput` | 0-100 | Saturation percentage |
| `LightnessInput` | 0-100 | Lightness percentage |

## Keyboard Navigation

- **Up/Down arrows**: Increment/decrement by 1
- **Shift + Up/Down**: Increment/decrement by 10
- **Enter**: Confirm the value

## Behavior

- Invalid values are reverted to the previous valid value
- Values are clamped to their respective ranges

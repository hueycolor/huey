---
title: Saturation Slider
description: Saturation channel slider
---

A slider component for controlling the saturation channel. Displays a gradient from grayscale to fully saturated based on the current hue.

## Value Range

- **Min**: 0 (grayscale)
- **Max**: 100 (fully saturated)
- **Step**: 1

## Accessibility

- `aria-label` defaults to `'Saturation slider'`
- Keyboard navigation with arrow keys

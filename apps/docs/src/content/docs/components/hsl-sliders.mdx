## Usage

<Tabs syncKey="framework">
  <TabItem label="Vue">
  ```ts
  import { HueSlider, SaturationSlider, LightnessSlider } from '@hueycolor/vue'
  ```
  ```vue
  <script setup>
  import { ref } from 'vue'
  import { HueyRoot, HueSlider, SaturationSlider, LightnessSlider, hueyColor } from '@hueycolor/vue'

  const color = ref(hueyColor('#1A3263'))
  </script>

  <template>
    <HueyRoot v-model="color">
      <HueSlider />
      <SaturationSlider />
      <LightnessSlider />
    </HueyRoot>
  </template>
  ```
  </TabItem>
  <TabItem label="Svelte">
  ```ts
  import { HueSlider, SaturationSlider, LightnessSlider } from '@hueycolor/svelte'
  ```
  ```svelte
  <script>
  import { HueyRoot, HueSlider, SaturationSlider, LightnessSlider, hueyColor } from '@hueycolor/svelte'

  let color = $state(hueyColor('#1A3263'))
  </script>

  <HueyRoot bind:color>
    <HueSlider />
    <SaturationSlider />
    <LightnessSlider />
  </HueyRoot>
  ```
  </TabItem>
</Tabs>

### Value Ranges

- **HueSlider**: 0-360 degrees
- **SaturationSlider**: 0-100%
- **LightnessSlider**: 0-100%

---
title: Alpha Input
description: Alpha channel input field
---

An input component for controlling the alpha (opacity) channel of the color. Accepts values from 0 to 100.

## Value Range

- **Min**: 0 (fully transparent)
- **Max**: 100 (fully opaque)

## Keyboard Navigation

- **Up/Down arrows**: Increment/decrement by 1
- **Shift + Up/Down**: Increment/decrement by 10
- **Enter**: Confirm the value

## Behavior

- Invalid values are reverted to the previous valid value
- Values are clamped to the 0-100 range

---
title: Red Input
description: Red channel input field
---

An input component for controlling the red channel of the RGB color model. Accepts values from 0 to 255.

## Value Range

- **Min**: 0
- **Max**: 255

## Keyboard Navigation

- **Up/Down arrows**: Increment/decrement by 1
- **Shift + Up/Down**: Increment/decrement by 10
- **Enter**: Confirm the value

## Behavior

- Invalid values are reverted to the previous valid value
- Values are clamped to the 0-255 range
- Automatically updates hue, saturation, and lightness values

---
title: Color Dropper
description: Eyedropper/color picker tool
---

A button component that activates the browser's native EyeDropper API, allowing users to pick any color from their screen.

::callout{icon="i-lucide-info"}
The EyeDropper API is only available in Chromium-based browsers. The component will not render if the API is unavailable.
::

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `error` | `Error` | Emitted when the eyedropper fails or is cancelled |

## Slots

| Slot | Description |
|------|-------------|
| `default` | Button content (icon, text, etc.) |

## Usage

```vue
<ColorDropper @error="handleError">
  <EyeDropperIcon />
</ColorDropper>
```

## Accessibility

The component uses proper ARIA attributes:
- `aria-label` defaults to `'Eye dropper button'`
- Inherits all button HTML attributes
